# AI question generation with GitHub Actions

This pack adds a GitHub Actions workflow that uses the repository secret `OPENAI_API_KEY` to generate `questions_ai.json`.

## Files

- `.github/workflows/generate_questions.yml` — manual/scheduled workflow
- `scripts/generate_questions.py` — OpenAI question generator
- `questions_ai.json` — generated by the workflow and committed back to the repo

## Setup

1. In GitHub, open your repository settings.
2. Go to **Secrets and variables → Actions**.
3. Add or confirm the repository secret named `OPENAI_API_KEY`.
4. Commit these files.
5. Open the **Actions** tab and run **Generate AI English Questions** manually.

## Notes

- The API key is only referenced inside the workflow as `${{ secrets.OPENAI_API_KEY }}`.
- The generated `questions_ai.json` is committed to the repository, so do not ask the model to include private information in generated content.
- If you do not want scheduled generation, remove the `schedule:` block from the workflow.
